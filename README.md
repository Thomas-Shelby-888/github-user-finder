# GitHub User Finder

**Автор:** Лапач Анатолий Игоревич

## Описание
GitHub User Finder — это GUI-приложение на Python (Tkinter), которое ищет пользователей GitHub через GitHub API, показывает результаты в списке и позволяет добавлять найденных пользователей в избранное. Избранные пользователи сохраняются и загружаются в JSON.

## Как использовать API
Приложение использует GitHub Search API для пользователей:
`https://api.github.com/search/users?q=...`

При вводе запроса приложение отправляет GET-запрос к API и получает список пользователей в формате JSON. Для работы с API в коде используется библиотека `requests`.

## Возможности
- Поиск пользователей GitHub по имени.
- Вывод результатов поиска в списке.
- Добавление пользователей в избранное.
- Сохранение и загрузка избранного в JSON.
- Проверка корректности ввода: поле поиска не должно быть пустым.

## Как использовать
1. Установите Python 3.
2. Установите зависимости:
   ```bash
   pip install requests
   ```
3. Запустите файл `github_user_finder.py`.
4. Введите имя пользователя и нажмите **Искать**.
5. Выберите результат и нажмите **Добавить в избранное**.
6. Используйте кнопки сохранения и загрузки JSON для избранного.

## Примеры тестов
- Ввести пустую строку — программа должна показать ошибку.
- Ввести `octocat` — должны появиться результаты.
- Добавить пользователя в избранное и сохранить список в JSON.
- Загрузить JSON обратно и проверить, что список восстановился.
